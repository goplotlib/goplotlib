goplotlib provides fourteen chart types, each optimised for a different kind of data.
Click a card to go to the full documentation page.

{{< cards >}}
  {{< card link="line" title="Line Chart" subtitle="Continuous data over time or a numeric axis. Supports smoothing, fills, dash patterns, and markers." >}}
  {{< card link="bar" title="Bar Chart" subtitle="Compare values across discrete categories. Supports single, grouped, and negative bars." >}}
  {{< card link="scatter" title="Scatter Plot" subtitle="Relationships between two numeric variables. Multiple marker shapes: circle, square, diamond, triangle." >}}
  {{< card link="step" title="Step Chart" subtitle="Piecewise-constant lines for CDFs, state transitions, and discrete events." >}}
  {{< card link="hbar" title="Horizontal Bar" subtitle="Bars oriented left-to-right — ideal for long category labels and rankings." >}}
  {{< card link="stacked_bar" title="Stacked Bar" subtitle="Part-to-whole breakdowns across categories with stacked coloured segments." >}}
  {{< card link="stacked_area" title="Stacked Area" subtitle="Cumulative totals over a continuous axis with filled, stacked bands." >}}
  {{< card link="histogram" title="Histogram" subtitle="Distribution of a numeric variable. Auto-bins via Sturges' rule or explicit edges." >}}
  {{< card link="bubble" title="Bubble Chart" subtitle="Scatter plot with a third variable encoded as circle radius." >}}
  {{< card link="heatmap" title="Heatmap" subtitle="Scalar matrix as a colour grid. Correlation matrices, activity calendars, confusion matrices." >}}
  {{< card link="pie" title="Pie & Donut" subtitle="Part-to-whole proportions as circle sectors. Optional donut cutout and segment explosion." >}}
  {{< card link="box" title="Box Plot" subtitle="Five-number summary (Q1, median, Q3, whiskers, outliers) for one or more groups." >}}
  {{< card link="error_bars" title="Error Bars" subtitle="Overlay ±uncertainty indicators on scatter or line charts." >}}
{{< /cards >}}
